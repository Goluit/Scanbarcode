What is a Barcode Scanner?

A barcode scanner is an input device that captures and reads information contained in a barcode. Traditional laser scanners read the reflection of light from the black and white lines. Our online scanner utilizes your device's high-definition camera and advanced ZXing decoding algorithms to read 1D barcodes (like UPC, EAN, Code 128) and 2D barcodes (like QR codes and Data Matrix) instantly.

What is a QR Code Scanner?

QR (Quick Response) codes are two-dimensional matrix barcodes that can hold thousands of characters of data. Unlike standard barcodes, QR codes can be scanned from any direction. Our tool supports all standard QR formats, including URLs, WiFi credentials, vCards, and plain text.

How Barcode Scanning Works

The process involves three main steps: illumination, reflection/conversion, and decoding. Our software illuminates the code with the camera flash (if needed), captures the image, and converts the light/dark areas into binary data. This binary string is then mapped to a character set based on the specific symbology (e.g., Code 128 set B).
